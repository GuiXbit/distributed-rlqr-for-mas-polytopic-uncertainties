# Distributed RLQR for Multi-Agent Systems with Polytopic Uncertainties

This repository contains codes being developed as part of my M.Sc. research in Electrical Engineering at the University of São Paulo (USP), São Carlos School of Engineering (EESC).

The main goal of this project is to investigate distributed consensus control for discrete-time multi-agent systems subject to uncertainties, with emphasis on recursive robust linear quadratic regulator (RLQR)-based formulations. In particular, this branch of the work focuses on the study of polytopic uncertainties and their incorporation into consensus-oriented control design.

The research explores formulations in which the control law is obtained from recursive optimization procedures, seeking alternatives to more traditional LMI-based approaches commonly found in the literature. The implementation aims to support both theoretical development and numerical validation of the proposed methods.

## Research Context

This project is part of my M.Sc. at USP and is related to:
- distributed control of multi-agent systems;
- consensus and tracking consensus problems;
- robust control under structured uncertainties;
- recursive Riccati-based formulations;
- numerical simulations in MATLAB.
